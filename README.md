> 🌐 **Smart Learning Hub** adalah platform belajar interaktif berbasis web untuk membantu siswa mempersiapkan Asesmen.
---

## 🎯 Fitur Utama

- 🧠 **Materi + Kuis** interaktif untuk setiap mata pelajaran.
- 🎮 Mode kuis dengan pilihan:
  - Tampilkan jawaban langsung
  - Tampilkan jawaban di akhir
- 🎯 **Progress bar** dan statistik skor & waktu.
- 🌙 **Dark/Light mode** toggle.
- 🎛️ **Navigasi dengan keyboard dan swipe (mobile friendly)**.
- 🔒 **Fitur anti-mencontek**: blok copy-paste, klik kanan, dan seleksi.
- 🧩 Modular: muat soal per mapel dari file JS eksternal (`mapel/*.js`).

---

## 🛠️ Teknologi yang Digunakan

- **Frontend:**  
  - HTML5  
  - CSS3 (custom styling + responsive layout)  
  - Vanilla JavaScript  
  - FontAwesome Icons  

- **Fitur JavaScript Tambahan:**
  - Transisi smooth antar halaman
  - Randomisasi soal dan jawaban
  - Timer, score tracker, dan difficulty tagging
  - Animasi & notifikasi dinamis
  - Mode tema (gelap/terang)

## 📂 Struktur Proyek

```

📁 smart-learning-hub/
├── index.html         # Halaman utama dengan UI
├── script.js          # Logika utama kuis dan navigasi
├── styles.css         # Gaya UI (tidak disertakan di sini)
└── mapel/
├── pkn.js
├── bindo.js
└── ...            # File soal per mata pelajaran

```

---

## 🚀 Cara Menjalankan

1. **Clone repo atau download ZIP** proyek ini.
2. Pastikan folder `mapel/` berisi file soal seperti `pkn.js`, `matematika.js`, dll.
3. Buka `index.html` di browser (offline mode sudah cukup).
4. Pilih mapel ➝ baca materi ➝ kerjakan kuis ➝ lihat hasilmu!
